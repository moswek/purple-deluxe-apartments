# Purple Deluxe Apartments - Project Blueprint

## Project Overview
**Brand Name:** Purple Deluxe Apartments (Purple Majesty Suites)
**Location:** Kyanja, Kampala, Uganda
**Tagline:** Experience Royal Indulgence. Your exclusive sanctuary in the heart of Kampala.
**Service Model:** Luxury Fully-Serviced B&B / Furnished Apartments for short and long-term stays.

## Core Features (Current)
- **Hero Section:** High-impact visual introduction with direct "Call to Book" CTA.
- **Our Legacy (About):** Narrative of luxury, comfort, and the tranquil Kyanja location.
- **Exquisite Suites:** Interactive carousel showcasing premium units:
  - *The Royal Sovereign*: Panoramic city views, ultimate luxury.
  - *The Golden Crest*: Bathed in warm gold tones and opulent design.
  - *The Amethyst Dream*: Spacious retreat with calming amethyst accents.
- **Amenities:** Showcase of world-class facilities (Wi-Fi, 24/7 Security, Parking, DSTV).
- **Explore Kampala (Location):** Strategic positioning in Kyanja with drive-time context to key Kampala landmarks.
- **Direct Booking:** Streamlined customer acquisition via direct phone contact (+256-780-PURPLE).

## Visual Identity & Style
- **Primary Color:** Deep Purple (#4A148C / HSL 271 76% 31%) - Evokes royalty.
- **Background:** Soft Lavender (#E6E6FA / HSL 255 60% 96%) - Elegant backdrop.
- **Accent:** Soft Gold (#D4AF37 / HSL 43 74% 49%) - Opulence and highlights.
- **Typography:**
  - Headlines: 'Belleza' (Humanist Sans-Serif).
  - Body: 'Alegreya' (Humanist Serif).
- **Components:** Glassmorphism headers, rounded-2xl cards, subtle shadows, and smooth parallax effects.

## Tech Stack
- **Framework:** Next.js 15 (App Router).
- **Styling:** Tailwind CSS + ShadCN UI.
- **Components:** Lucide Icons, Embla Carousel, Radix UI Primitives.
- **Deployment:** Firebase App Hosting.

## Navigation Strategy
- Single-page application architecture using hash-based navigation (#about, #suites, #amenities, #location).
- Fixed glassmorphism header that transitions from transparent to solid on scroll.

## Booking Protocol
- All "Book" actions trigger a direct phone call to `+256780876369`.
- No automated forms or AI concierge are currently active on the frontend.
